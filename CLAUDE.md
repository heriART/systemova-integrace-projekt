## Project Overview

This is a **semester project** for the System Integration course at Czech University of Life Sciences Prague (ČZU). The project designs the integration of **Salesforce CRM** into a fictional hotel chain **Aurora Hotels a.s.** (7 hotels across Czech Republic). The system integrator is **Seyfor a.s.** (formerly Solitea).

## Repository Structure

- `data/projekt.docx.md` — main project document (Markdown export from .docx)
- `data/Vzorový-projekt-Systémová-integrace.pdf` — sample/template project for reference
- `data/Zadani projektu.pdf` — project assignment specification

There is no source code — this is a document-only project.
